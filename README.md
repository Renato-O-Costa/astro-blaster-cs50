# astro-blaster-cs50
Neste jogo, feito no Scratch, uma espaçonave deve destruir os meteoros.

---

## Sobre o projeto - # Astro Blaster

O **Astro Blaster** é um jogo de nave espacial estilo arcade desenvolvido no Scratch para o projeto da Semana 0 do curso CS50 da Harvard. O objetivo do jogador é defender o espaço destruindo os meteoros que caem continuamente do topo da tela.

---

## 🎮 Funcionamento e Controles

O jogo consiste em controlar a nave na base da tela, desviar dos obstáculos e disparar lasers para causar explosões ao colidir com os meteoros.

* **Setas Direita / Esquerda:** Movem a nave horizontalmente pelo cenário.
* **Barra de Espaço:** Dispara o laser a partir da posição atual da nave.

---

## 🚀 Como Jogar (Via Download do Código)

Como este projeto está armazenado em formato de código fonte do Scratch (`.sb3`), você pode carregá-lo e jogá-lo diretamente no seu navegador seguindo os passos abaixo:

1. **Baixe o arquivo do jogo:**
   * Clique no arquivo `astro_blaster.sb3` aqui no repositório do GitHub.
   * Clique no botão **Download** (ou Baixar) para salvar o arquivo no seu computador.

2. **Abra o visualizador do Scratch:**
   * Acesse o editor online oficial do [Scratch MIT](https://scratch.mit.edu/projects/editor/) ou o [TurboWarp](https://turbowarp.org/).

3. **Carregue o projeto:**
   * No menu superior do site escolhido, clique em **Arquivo** (File).
   * Selecione a opção **Carregar do seu computador** (Load from your computer).
   * Escolha o arquivo `astro_blaster.sb3` que você baixou.

4. **Divirta-se:**
   * Clique na **Bandeira Verde** 🟩 acima do palco para iniciar o jogo e clique no **Botão Vermelho** 🛑 para parar. Você também pode visualizar e editar todos os blocos de código!
